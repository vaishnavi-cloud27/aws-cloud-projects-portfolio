# Static Website Hosting with S3 & CloudFront

Hosted a static website on AWS S3 and secured it using CloudFront CDN.

## Key Features
- S3 Static Website Hosting
- CloudFront Distribution Setup
- Restricted direct S3 access
- Improved performance with caching

## Tools Used
AWS S3, CloudFront, HTML
