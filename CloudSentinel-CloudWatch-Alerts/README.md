CloudSentinel is a monitoring and alert system built using AWS CloudWatch and SNS.

## Project Goal
Send an email alert when EC2 Memory or Disk usage exceeds thresholds.

## What I Implemented
- Installed CloudWatch Agent on EC2
- Collected memory and disk metrics
- Created CloudWatch Alarms
- Integrated SNS for email notifications

## Tools Used
AWS EC2, CloudWatch, CloudWatch Agent, SNS, Linux
